# Recura

Recura ist eine eigenständige One-Page-Business-Website für KI-Automatisierung im Mittelstand.

## Stack
- Next.js
- React
- Tailwind CSS

## Projektstatus
Die erste Website-Grundlage ist implementiert, inklusive:
- Hero
- Leistungen
- Use Cases
- Über uns
- Vorgehen
- Kontakt
- Impressum
- Datenschutz

## Wichtige Projektdateien
- `RECURA_PLAN.md`
- `RECURA_TODO.md`

## Entwicklung
```bash
npm run dev
```

## Qualitätssicherung
```bash
npm run lint
npm run build
```
